üìñ How to Use (Step-by-Step)
Follow these steps to resize your images:

1. Prepare your images Create a folder (e.g., input_images) and put all the images you want to resize inside it.

2. Run the script Open your terminal inside the project folder and type: python bulk_resizer_lite.py


3. Follow the prompts The script will ask you for:

The path to your Source Folder (where your images are).

The path for the Destination Folder (where you want the resized images to go).

The desired Size (width/height or percentage).

4. Done! The script will process your images instantly. You will find the resized versions in your destination folder.
